Core request flow (general chat)
────────────────────────────────────────────────────────────────────────────────

  popup.js
    → externallib.php :: send_message()
    → fastapi_client.php :: send_chat()
    → api/routes_chat.py :: chat()
    → services/chat_service.py :: answer_question()
         → infrastructure/ollama_client.py :: get_embedding(), chat_completion()
         → retrieval/chroma_store.py :: ensure_course_indexed(), chroma_search()
         → prompts/chatbot_prompts.py :: build_system_prompt()
    → routes_chat.py :: _finish() → JSON ChatResponse
    → Moodle saves message → popup.js displays reply

  Full architecture: see ../architectural.txt
